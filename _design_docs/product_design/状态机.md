# 状态机管理文档 (State Machine Management)

系统状态由用户显式操作推进。推进方式分为两种：
- **查看模式**（PhaseSelector 下拉切换）：自由切换查看范围，不创建快照，不修改状态机
- **推进模式**（ConfirmButton 点击确认）：创建快照 → 推进 `currentPhase` → 锁定前一阶段

推进模式严格单向流转。Orchestrator 的行为完全由当前 `current_phase` 决定。

## 状态定义与流转规则

### 1. `STATE_1_CONCEPT` (概念与设定收集)
*   **触发条件**：系统初始化。
*   **Orchestrator 行为**：与用户多轮对话，提炼创意。直接生成/更新 `worldbuilding.md`, `characters.md`, `plot_synopsis.md`。
*   **前端 UI**：
    *   右侧栏可见卡片：worldbuilding.md / characters.md / plot_synopsis.md
    *   左侧栏：显示选中卡片的上次确认版本（首次为空）
    *   中间栏：显示该卡片当前最新内容
    *   确认按钮：「确认设定，进入大纲构建」（仅三个文件都存在时启用）
*   **快照时机**：用户点击确认按钮时，对 3 个文件分别生成 `*.approved.md` 快照。
*   **流转条件**：用户在前端点击 **【确认设定，进入大纲构建】** 按钮 $\rightarrow$ 状态变更为 `STATE_2_MACRO`。

### 2. `STATE_2_MACRO` (宏观结构构建)
*   **触发条件**：进入阶段二。前端展示 `act_map.md` 和 `sequence_list.md` 的资产卡片。
*   **Orchestrator 行为**：拦截用户对话，提取”修改指令”。将 3 个基础 MD + 修改指令注入给 **Agent A**。接收并更新幕地图和序列清单。
*   **前端 UI**：
    *   右侧栏新增可见卡片：act_map.md / sequence_list.md
    *   阶段一卡片（worldbuilding/characters/plot_synopsis）标记为锁定（只读，可点击查看但不可编辑）
    *   确认按钮：「确认大纲，进入场景细化」
*   **快照时机**：用户点击确认按钮时，对 act_map.md / sequence_list.md 生成快照。
*   **流转条件**：用户在前端点击 **【确认大纲，进入场景细化】** 按钮 $\rightarrow$ 状态变更为 `STATE_3_MICRO`。

### 3. `STATE_3_MICRO` (微观结构精铸)
*   **触发条件**：进入阶段三。前端展示 `scene_beat_outline.md` 资产卡片。
*   **Orchestrator 行为**：拦截用户对话，提取”修改指令”。将 5 个上游 MD + 修改指令注入给 **Agent B**。接收并更新场景-节拍整体大纲。
*   **前端 UI**：
    *   右侧栏新增可见卡片：scene_beat_outline.md
    *   阶段一、阶段二卡片全部标记为锁定
    *   确认按钮：「完成创作」
*   **快照时机**：用户点击确认按钮时，对 scene_beat_outline.md 生成快照。
*   **流转条件**：用户在前端点击 **【完成创作】** $\rightarrow$ 状态变更为 `STATE_COMPLETED`。

### 4. `STATE_COMPLETED` (交付与锁定)
*   **触发条件**：创作完成。
*   **Orchestrator 行为**：锁定所有 MD 文件的写入权限，提供最终打包导出。
*   **前端 UI**：
    *   所有 6 个卡片可见但全部锁定（只读）
    *   左侧栏 / 中间栏仅作对比查看，不可编辑
    *   对话输入框禁用，显示「创作已完成」
    *   文件导入区禁用
    *   确认按钮禁用，显示「已完结」
    *   显示「导出最终文件」按钮
    *   显示「重新创作」按钮（灰色，点击弹出二次确认对话框）
*   **回退行为**：`STATE_COMPLETED` 允许回退。回退途径有两种：
    1. 用户点击 **「重新创作」按钮** → 直接弹出二次确认对话框（无需对话输入）
    2. 意图提纯检测到"推翻核心设定"意图（仅当用户通过其他方式输入时）
    *   二次确认对话框内容："这将清空所有已创作内容（含基础设定文件），恢复至概念设定阶段。确定继续？"
    *   用户确认后执行回退流程：清空所有 Agent writes 文件 + Phase 1 直接产出文件（worldbuilding.md / characters.md / plot_synopsis.md）+ 对应 `*.approved.md` 快照 → 解除所有锁定 → 跳回 `STATE_1_CONCEPT`
