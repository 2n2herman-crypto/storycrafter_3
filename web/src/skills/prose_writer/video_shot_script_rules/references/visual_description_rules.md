# 视觉描述规则

## 视觉转化五大铁律

1. **可拍性原则**：所有描述必须是"摄像机能拍到的"。禁止"他的内心涌起一阵复杂的情绪"——这类内容无法被镜头捕获。改为"他的眼角抽搐了一下，拳头在桌面下握紧又松开"。

2. **空间连续原则**：同一场景内的镜头间，人物位置、道具位置必须保持空间连续。上一个镜头角色在窗前，下一个镜头突然变成在沙发上——除非中间有"走过去"的镜头承接。

3. **视觉焦点原则**：每个镜头只承载一个视觉焦点。不要在一个镜头描述里堆叠三个以上视觉元素。复杂场景用多镜头拆解。

4. **运动驱动原则**：用运镜来描述情绪，不是用形容词。不是"悲伤的镜头"，而是"镜头缓缓向后退，将角色越缩越小地困在门框里"。

5. **声画分离原则**：台词/音效与画面描述分开标注。声音有它自己的叙事线。

## 四维度描述规范

对每个镜头，以下四维度依次填写（每个维度一句即可，不要过度展开成段）：
1. **主体**：谁/什么在画面中（含位置关系）
2. **动作**：主体在做什么
3. **构图**：景别+角度+画面构成
4. **质感**：光影/色调/氛围（仅在特殊时才写，日常场景可省略）
