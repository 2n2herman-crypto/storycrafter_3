# 时长估算公式

## 基础公式

每个镜头的预估时长由以下要素加总计算：

### 台词时长
- **公式**：台词字数 ÷ 8 字/秒 + 1 秒（缓冲）
- 无台词镜头 = 0

### 动作时长
- 简单动作（拿、放、转头）= 1-2 秒
- 中等动作（走几步、坐下）= 2-4 秒
- 复杂动作（起身+走到窗边+向外看）= 4-6 秒
- 战斗/追逐类动作：按具体分解的子动作数 × 1 秒

### 叙事留白
- 情绪停顿/沉默 = 1-3 秒（仅关键情绪节点才加）
- 场景过渡 = 2 秒

## 标注格式

每镜头末尾标注计算依据：
- `预估时长：3s（台词10字÷8+1=2s + 简单动作1s）`
- `预估时长：7s（无台词0s + 复杂起身动作5s + 场景过渡2s）`

## 总量参考

- 一个 30 分钟短剧集：约 30-40 个镜头
- 一个 5 分钟短视频段落：约 8-12 个镜头
- 镜头平均时长：30-60 秒（含台词的动作镜头偏长，纯动作镜头偏短）
